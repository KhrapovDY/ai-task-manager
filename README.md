# AI Task Manager

Учебное веб-приложение для управления задачами.

## Планируемые возможности

- создание задачи;
- просмотр списка задач;
- изменение статуса задачи;
- удаление задачи;
- экспорт задач из PostgreSQL в CSV.

## Стек

- React + JavaScript;
- Node.js + Express;
- PostgreSQL;
- Python;
- Docker Compose;
- Git.
